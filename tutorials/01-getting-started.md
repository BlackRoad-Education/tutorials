# Getting Started with BlackRoad OS

## Prerequisites
- macOS or Linux
- Git, Node.js 20+, Python 3.11+
- GitHub account with access to BlackRoad-OS orgs

## Install the br CLI
```bash
git clone https://github.com/BlackRoad-OS/blackroad-os
cd blackroad-os
chmod +x scripts/br
export PATH="$PATH:$(pwd)/scripts"
```

## Your First Agent
```bash
br agent list         # List available agents
br cece whoami        # Check CECE identity
br git commit         # Smart AI commit
```
